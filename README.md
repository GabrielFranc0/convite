# Convite Digital — 50 Anos

Convite interativo para festa de 50 anos, hospedado no GitHub Pages.

## Como personalizar

Edite o arquivo `index.html` e altere:

| Campo | Onde alterar |
|---|---|
| Data, horário, traje, descrição | Seção `.info-grid` |
| Local no Google Maps | Link do botão verde (`href` do botão 📍) |
| WhatsApp para confirmação | Link do botão verde claro (`wa.me/...`) |
| Música de fundo | Adicione um arquivo `musica.mp3` na raiz do projeto |

### Link do Google Maps

Substitua o endereço no link:
```
https://maps.google.com/?q=SEU+ENDEREÇO+AQUI
```

### WhatsApp

Substitua o número (com código do país, sem +):
```
https://wa.me/5511999999999?text=Mensagem+personalizada
```
